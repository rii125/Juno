**Juno is a new language like Java.** Java is an object-oriented language, and Juno is influenced by it.
It also excels in type maintenance, drawing inspiration from the approach to writing type annotations seen in TypeScript.

```java
public static main() {
    print("Hello Juno!");
}
```

# What is Juno?
A concise introduction: “Juno is a new programming language designed with simplicity, type safety, and syntax beauty in mind.”

# Features
- Syntax inspired by Java (object-oriented clarity)
- Type safety influenced by TypeScript 
- AST/IR design for extensibility 
- Clear error handling and hints
